# Todo List App

This is a simple Todo List application created using React, Node.js, Express, MongoDB, and Tailwind CSS.

## Features

- Add tasks with titles and descriptions
- Mark tasks as completed
- View a list of tasks
- Delete tasks
  -Update task details

## Technologies Used

- React
- Node.js
- Express
- MongoDB
- Tailwind CSS
